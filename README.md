# 2D-TopDown-Game
2D game created using Unity, survive waves to complete it! 

Play the game by clicking [HERE](https://dmartinm95.github.io/2D-TopDown-Game/)!

**Controls:** 
- Move with WASD keys 
- Shoot by holding or pressing the Right Mouse Button (RMB)
